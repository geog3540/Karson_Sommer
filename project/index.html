Iowa Roads

Data from Iowa Department of Transportation Road Access Management System (RAMS), maps by Karson Sommer.

The roads first have county information joined in QGIS and the 4 name fields reduced to 1 name field. Then, the summary statistics are extracted from road network using SQL queries in Azure Databricks. (Query) A related analysis instead focusing on crash statistics can be found here: Presentation

A summary of which suffixes are the most common in each county can be found here: Summary Map

These maps display the proportion of roads in Iowa broken down by county. Change the suffix to look for a specific type of road. Selecting length looks at the percent of total miles in each county with the specified suffix. Selecting vehicle miles looks at the percent of total miles driven by the specified suffix. Disable the load screen to more easily toggle between length and vehicle miles.
